Blog Post Five
12/06/2023
This assignment was by far the most difficult one for me, and the one I have spent the most amount of time on. I was generously offered an extension to the deadline for this assignment because I couldn't get it working by the original deadline. As of now, I’ve worked through the issues I had been having with handlebars and now I am very happy with how the assignment has turned out. I struggled for a very long time to get handlebars and express-handlebars working in my project. I was eventually able to get it right by rewatching a couple lectures more than once. Once I was able to get handlebars working the way I expected it to, I was able to make a lot of progress on the assignment. I was at first very intimidated by the fact that we would be marked down for duplicate HTML code in our handlebars templates, but by the end of the assignment I had a strong enough understanding to tackle that. As far as I am aware, there is no duplicate HTML code in any of the handlebars files and I have been able to meet every requirement for the assignment. I certainly was able to beat my own expectations for this assignment by the time I finished it, which is very nice.
